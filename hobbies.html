John Chiodo's Hobbies

I like to build home PCs, my latest was a NAS (wiki: Network Attached Storage), which I am using to store media, PC backups, etc.

You can purchase a compeletly working NAS off the shelf, but what is the fun in that? There are many options if you want to make one yourself. I choose UNraid from lime-technologies.

Main reasons I chose Unraid:

  • Can run on older hardware.
  • Can expand easily with different size drives.
  • Supports single and dual parity protection.
  • Support virtualization and Docker containers.

Other future uses for my UNraid NAS:

  • Virtualize Win10 and CentOS linux box for general websurfing, etc.
  • Virtualize a Plex Media Server so I can watch my media content on any network capable device in the house.
  • Virtualize a Win10 box with graphics card for PC gaming.
